Detailed Description

This class sets up a HTTP (not HTTPS!) REST server for computing routes.

Acceptable requests: GET server_addr/compute?start=lat,lon,h&dest=lat,lon,h where lat and lon are latitude and longitude respectively and h is the elevation above sea level in meters. This returns a unique identifier that needs to be saved for later to get the finished route.

GET server_addr/retrieve?id=unique where unique is the identifier returned by compute/ This will either return the computed route's control points or false
