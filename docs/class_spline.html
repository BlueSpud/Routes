Performs De Boor's algorithm recursively to evaluate a B-Spline. Algorithm can be found at https://en.wikipedia.org/wiki/De_Boor%27s_algorithm There appears to be different representation of variables depending on what article you are reading. The parameters here match the ones outlined in the Wikipedia article.

Parameters
iThe i value in De Boor's algorithm according to the Wikipedia article.
rThe r value in De Boor's algorithm according to the Wikipedia article.
pThe degree of the B-spline.
controlsThe control points that make up the B-Spline.
knotsThe knot vector.
tThe parametric parameter of the B-spline curve. Should be in the range [*knots.begin(), *knots.end()].
