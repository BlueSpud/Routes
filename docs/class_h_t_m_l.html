Detailed Description

This class is responsible for creating html output for a calculated path. It reads template.html out of the htdocs folder and creates an index.html. This index.html file needs to be hosted with a local web server on port 8888. This is a requirement of MapBoxJS for loading the stylesheet that we are using.
