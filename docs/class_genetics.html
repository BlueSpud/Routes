This function runs the genetic algorithm to compute the most efficient path from start to dest.

Parameters
popThe population of individuals that represent solutions to the path. This should be a new population but an old one can be reused assuming that it has been simulated with the same start and dest.
podThe information about the hyperloop pod. Determines how curved the track can be.
Returns
The points of the calculated path in meters. This will need to be converted to latitude and longitide to be properly displayed.
