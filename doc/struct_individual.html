Detailed Description

Individual is a convenience so that individuals can be treated as units rather than as a single float vector, which is how they are stored. Individual is not used to store any new data, but rather reference other data.

Member Data Documentation

◆ genome

glm::vec4* Individual::genome

The pointer to the genome of the individual. This is an array of glm::vec4 with length equal to num_genes.

◆ header

glm::vec4* Individual::header

A reference to this individual's header. The header contains the computed cost of the individual, so long as it has been downloaded from the GPU. Cost is stored in the X component and Y, Z and W are currently unused.

The length of the genome (num glm::vec4's). All individuals in the same population have the same number of genes.
